Conways Spiel des Lebens

Ein Zelluläres Automaton - Klicke auf Zellen, um sie lebendig zu machen

Wie funktioniert das Spiel?

Regeln:

  • Eine lebende Zelle mit 2-3 Nachbarn überlebt
  • Eine lebende Zelle mit weniger als 2 Nachbarn stirbt
  • Eine lebende Zelle mit mehr als 3 Nachbarn stirbt
  • Eine tote Zelle mit genau 3 Nachbarn wird lebendig

Bedienung:

  • Klick auf Zellen: Toggle zwischen lebendig/tot
  • Start: Simulation ausführen
  • Step: Einen Generationenschritt vorwärts
  • Renew: Zufälliges Gitter generieren
  • Clear: Gitter zurücksetzen
